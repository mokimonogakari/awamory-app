# カクテルレコメンド 判定ロジック仕様書

## 概要

ユーザーの入力（度数嗜好・スタイル・シーン・フード・天気）をもとに、各カクテルにスコア（0〜100点）を付与し、上位5件をレコメンドする。

## 入力要素

| # | 入力項目 | 選択肢 | デフォルト |
|---|---------|--------|-----------|
| 1 | 度数の好み（strength） | 弱い / 普通 / 強い | 普通 |
| 2 | カクテルスタイル（volume） | ロング / ショート | ロング |
| 3 | シーン（scene） | 会食 / 夫婦 / カップル / 女子会 / 男子会 / 未選択 | 未選択 |
| 4 | フードペアリング（foods） | 20種から複数選択可 | 未選択 |
| 5 | 気温（temp） | 10〜40°C スライダー | 25°C |
| 6 | 湿度（humidity） | 20〜100% スライダー | 60% |

## スコアリングアルゴリズム

### ベーススコア: 50点

全カクテル共通の初期スコア。

---

### 1. 度数スコア（最大+30点 / 最小-10点）

泡盛の使用量（awamoriMl）と味わい（taste）で判定。

#### 「弱い」選択時
| 条件 | スコア |
|------|--------|
| 泡盛 ≤ 30ml | +20 |
| 泡盛 ≤ 35ml | +10 |
| 泡盛 > 35ml | -10 |
| 甘口 or スウィート | +10 |

#### 「強い」選択時
| 条件 | スコア |
|------|--------|
| 泡盛 ≥ 45ml | +20 |
| 泡盛 ≥ 40ml | +15 |
| 泡盛 < 40ml | -5 |
| 辛口 or ドライ | +10 |

#### 「普通」選択時
| 条件 | スコア |
|------|--------|
| 30ml ≤ 泡盛 ≤ 40ml | +10 |
| ミディアム or 中口 | +10 |

---

### 2. スタイルスコア（最大+20点）

| 条件 | スコア |
|------|--------|
| 「ロング」選択 かつ ロングカクテル | +20 |
| 「ショート」選択 かつ ショートカクテル | +20 |

---

### 3. シーンスコア（最大+23点）

#### カップル / 夫婦
| 条件 | スコア |
|------|--------|
| スウィート / 甘口 / やや甘口 | +12 |
| 説明にロマンチック系キーワード含む | +8 |

#### 女子会
| 条件 | スコア |
|------|--------|
| スウィート / 甘口 | +15 |
| 女性向けキーワード（花、可愛い等） | +8 |
| 泡盛 ≤ 30ml | +5 |

#### 男子会
| 条件 | スコア |
|------|--------|
| ドライ / 辛口 | +12 |
| 泡盛 ≥ 40ml | +8 |
| 常圧蒸留 | +5 |

#### 会食
| 条件 | スコア |
|------|--------|
| ミディアム / 中口 | +10 |
| 受賞歴あり | +10 |

---

### 4. フードペアリングスコア（最大+38点）

選択されたフードの**平均味の濃さ**と個別属性で判定。

#### 味の濃さ（平均）による判定
| 条件 | スコア |
|------|--------|
| 平均 ≥ 3.5 かつ 常圧蒸留 | +12 |
| 平均 ≥ 3.5 かつ 辛口/中口 | +8 |
| 平均 ≤ 2.0 かつ 減圧蒸留 | +10 |
| 平均 ≤ 2.0 かつ 甘口 | +5 |

#### 蒸留タイプマッチング
| 条件 | スコア |
|------|--------|
| フードが常圧蒸留推奨 かつ カクテルが常圧蒸留 | +8 |
| フードが減圧蒸留推奨 かつ カクテルが減圧蒸留 | +8 |

#### 特殊ペアリング
| 条件 | スコア |
|------|--------|
| 甘いフード（sweet ≥ 4）かつ ドライカクテル | +8 |
| 油っぽいフード（oil ≥ 4）かつ 炭酸カクテル | +10 |

**注意**: フード複数選択時、常圧・減圧両方の推奨フードが含まれる場合、両方のボーナスが加算される可能性がある（最大+16点）。

---

### 5. 天気スコア（最大+20点）

#### 気温
| 条件 | スコア |
|------|--------|
| 30°C以上 かつ 炭酸 | +10 |
| 30°C以上 かつ ロング | +5 |
| 30°C以上 かつ 減圧蒸留 | +5 |
| 18°C以下 かつ 常圧蒸留 | +8 |
| 18°C以下 かつ 非炭酸 | +5 |
| 18°C以下 かつ 辛口 | +5 |

#### 湿度
| 条件 | スコア |
|------|--------|
| 75%以上 かつ 炭酸 | +8 |
| 75%以上 かつ 辛口 | +5 |

---

### 6. 受賞ボーナス（最大+5点）

| 条件 | スコア |
|------|--------|
| 最優秀賞 | +5 |
| 知事賞 | +3 |

---

## 最終スコア計算

```
finalScore = clamp(0, 100, baseScore + 各カテゴリのスコア合計)
```

理論上の最大スコア: 50 + 30 + 20 + 23 + 38 + 20 + 5 = **186点** → 100点にクランプ

---

## 泡盛銘柄レコメンド

スコア1位のカクテルの蒸留タイプ（常圧/減圧）に一致する銘柄を最大5件表示。

---

## ペアリングティップ

フード選択時に自動生成。平均味の濃さに応じた文章 + 気温による追加アドバイス。

| 平均味の濃さ | メッセージ |
|------------|----------|
| ≥ 3.5 | 常圧蒸留推奨 |
| ≤ 2.0 | 減圧蒸留推奨 |
| その他 | どちらでも |

---

## データ構造

### カクテルデータ (COCKTAILS)
```
{
  id, name, nameEn, desc,
  category: "ショートカクテル" | "ロングカクテル",
  taste: "ドライ（辛口）" | "ミディアム..." | "スウィート（甘口）",
  ingredients: "材料名 : 量;...",
  distill: "常圧蒸留" | "減圧蒸留",
  glass, recipe, award,
  awamoriMl: number,
  carbonated: boolean
}
```

### フードデータ (FOOD_DATA)
```
{
  name, intensity: 1-5, type,
  salt: 1-5, sweet: 1-5, sour: 1-5, oil: 1-5,
  recDistill: "常圧蒸留" | "減圧蒸留" | "どちらでも"
}
```

### 泡盛銘柄データ (AWAMORI_BRANDS)
```
{
  brand, maker,
  distill: "常圧蒸留" | "減圧蒸留",
  degree: number, note
}
```
