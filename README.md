# 泡盛カクテルレコメンダー (Awamory Cocktail Recommender)

## 概要

ユーザーの好み（度数・スタイル・シーン・フード・天気）をもとに、36種類の泡盛カクテルからスコアリングでベスト5をレコメンドするWebアプリ。カクテルごとに最適な泡盛銘柄もサジェストし、推定アルコール度数やこだわりポイントも表示する。

**デモ**: https://mokimonogakari.github.io/awamory-app/

本アプリは [Awamori open data 3/14](https://ool.connpass.com/event/382912/) ハッカソンで制作されました。バーテンダーが泡盛を使ったカクテルの作り方や最適な銘柄を知ることで、泡盛の魅力を沖縄県内外、さらには海外にも広く普及させることを目指しています。

## 技術スタック

- **フロントエンド**: HTML / CSS / Vanilla JavaScript（フレームワーク不使用）
- **ホスティング**: GitHub Pages（静的配信）
- **フォント**: Playfair Display + Noto Sans JP（Google Fonts）
- **データソース**: Google Spreadsheet（泡盛カクテルコンペティション受賞レシピ + 蒸留所情報）

## セットアップ

```bash
git clone https://github.com/mokimonogakari/awamory-app.git
cd awamory-app
# 静的サイトのためビルド不要。ローカルサーバーで開く:
python3 -m http.server 8000

```

## 機能一覧

| # | 機能 | 説明 |
|---|------|------|
| 1 | カクテルレコメンド | 6要素のスコアリング（0〜100点）で上位5件を表示 |
| 2 | 泡盛銘柄サジェスト | カクテルごとに最適な泡盛を3銘柄推薦（蒸留タイプ＋度数嗜好で判定） |
| 3 | アルコール度数推定 | 泡盛量 × 銘柄度数 ÷ 総容量で推定アルコール%を算出 |
| 4 | フードペアリング | 20種のフードから味の濃さ・蒸留タイプとのマッチングを判定 |
| 5 | 天気対応 | 気温・湿度スライダーで季節に合ったカクテルを優先 |
| 6 | シーン対応 | 会食・カップル・女子会・男子会などシーン別の最適化 |
| 7 | こだわり情報表示 | 各蒸留所の歴史・特徴・公式サイトリンクを表示 |

## データソース

- **カクテルレシピ**: 泡盛カクテルコンペティション受賞作品（総務大臣最優秀賞・沖縄県知事賞含む）
- **泡盛銘柄**: 沖縄県内20蔵元の代表銘柄（常圧蒸留10銘柄・減圧蒸留10銘柄）
- **フードデータ**: 沖縄料理を中心とした20品目の味覚属性データ

## スコアリングロジック

詳細は [docs/scoring-logic.md](docs/scoring-logic.md) を参照。

**概要**: ベーススコア50点 + 6カテゴリの加算/減算 → 0〜100点にクランプ

| カテゴリ | 最大加点 | 判定基準 |
|---------|---------|---------|
| 度数スコア | +30 | 泡盛使用量 × 味わいタイプ |
| スタイル | +20 | ロング/ショートの一致 |
| シーン | +23 | シーン別の味覚・度数マッチ |
| フードペアリング | +38 | 味の濃さ・蒸留タイプ・特殊ペアリング |
| 天気 | +20 | 気温・湿度による炭酸・蒸留タイプ調整 |
| 受賞ボーナス | +5 | 最優秀賞・知事賞 |

## 今後の課題

- **フードペアリングロジックのブラッシュアップ**: 現在のマッチングは味の濃さ・蒸留タイプ等の数値データに基づいているが、実際の料理との相性はプロのシェフ・料理人の知見が不可欠。シェフの監修により、より実践的で説得力のあるペアリング提案が可能になる
